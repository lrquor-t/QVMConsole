- **启动编排与状态恢复**：`main.go` 按序执行配置加载、数据库初始化、Libvirt RPC 连接建立，并在路由启动前通过 `Ensure*` 系列函数同步 OVS、VPC、端口转发等底层网络设施的运行态。
- **异步任务中心**：在 `main.go` 中集中注册数十种长耗时任务（如克隆、迁移、快照）到 `taskqueue`，实现请求快速响应与后台执行的解耦。
- **循环依赖解耦**：采用 `service/hooks_init.go` 中的 `init()` 机制，将根包函数注入子包（如 `vm/memory`）的全局 Hook 变量，在不引入循环导入的前提下实现跨层能力调用。
- **统一网关与静态托管**：`router/router.go` 定义全量 API 契约并挂载鉴权、限流中间件，同时通过 `NoRoute` 回退机制支持前端 SPA 路由，实现前后端一体化部署。

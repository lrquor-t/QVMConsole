- **启动编排与状态恢复**：`main.go` 负责引导服务，依次初始化配置、数据库、Libvirt RPC 连接，并在路由启动前执行网络桥接、端口转发及安全组的运行时状态恢复。
- **异步任务中心**：集成 `taskqueue` 模块，在启动时注册涵盖 VM 生命周期（克隆、迁移、快照）、存储管理及网络配置的数十种处理器，实现耗时操作的后台化与进度追踪。
- **分层架构与依赖解耦**：采用 Handler-Service-Model 分层设计，通过 `service/*_wire.go` 中的全局钩子变量解决核心包与子包间的循环导入问题，确保业务逻辑的模块化隔离。
- **统一网关与中间件链**：`router` 模块集中定义 API 路由，挂载 JWT 认证、RBAC 权限校验、速率限制及 CORS 等全局中间件，并支持前端 SPA 静态资源的回退处理。
